১. প্রথমে আপনার কম্পিউটারে Node.js ইনস্টল করুন: https://nodejs.org

২. ZIP ফোল্ডার আনজিপ করুন

৩. টার্মিনাল বা কমান্ড প্রম্পট খুলে backend ফোল্ডারে যান

৪. নিচের কমান্ডগুলো দিন:
   npm install
   node index.js
